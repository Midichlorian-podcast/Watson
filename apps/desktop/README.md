# apps/desktop — Tauri (placeholder)

Desktopová aplikace (Tauri obal `apps/web`) je naplánovaná na **v2**
(viz `files/funkcni_specifikace_v2_build_ready.md` §7).

Rust toolchain je už nainstalovaný (`cargo`), takže až přijde čas:

```bash
pnpm create tauri-app@latest
# nebo přidat src-tauri do tohoto adresáře a nakonfigurovat na build z ../web
```

Zatím záměrně prázdné, aby scaffold MVP zůstal štíhlý.
