06

Vyřešený barevný model

tři nezávislé kanály — puntík rušíme
Projekt = proužek
Levý barevný proužek karty nese identitu projektu. Výchozí = barva projektu; každý si ji přebarví jen pro sebe (projekt i konkrétní úkol). Pořadí: osobní barva úkolu → osobní barva projektu → výchozí.
Priorita = vlajka
Priorita se stěhuje z barvy řádku na vlajku-glyph (P1 plná, P4 žádná). Tím se barevný kanál uvolní pro projekt a obě věci se přestanou bít.
Puntík = zrušen
Tečka projektu byla redundantní — proužek ji nahrazuje. Hotovo zůstává: přeškrtnutí + ztlumení. Cizí osobní barvu nikdy nevidíš, vždy jen svou.
07

V kontextu

řádky + lišta zakládání
Úkoly — barva podle projektu, priorita vlajkou
Přidat úkol — input + ikonový řádek
Zavolat účetní ohledně faktur…
Nenastavená ikona je ztlumená; nastavená nese badge s hodnotou. Ťuk → drobný popover. Na mobilu se řádek vysune nad klávesnici.
Schvaluješ tento směr?
Když řekneš ano, zapojím set do FieldKitu a postupně ho rozjedu napříč aplikací (menu, modaly, řádky, kalendář) — a zároveň postavím barevný model: proužek = projekt, vlajka = priorita, puntík pryč.
